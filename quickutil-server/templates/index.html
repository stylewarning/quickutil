What's Quickutil?

Quickutil is a completely new way for organizing and distributing Common Lisp utilities. Read about why we made it or download the beta and try it for yourself.

  • 1. Install Quickutil

                > (ql:quickload :quickutil)
  • Find utilities

    2. Load Utilities

                > (qtlc:utilize{\n}
            {sp}{sp}{sp}{sp}{sp}{sp}:utilities '(:iota :riffle){\n}
            {sp}{sp}{sp}{sp}{sp}{sp}:categories '(:alexandria){\n}
            {sp}{sp}{sp}{sp}{sp}{sp}:symbols '(:split-sequence-if-not))

  • 3. Use Them

                > (qtl:riffle (qtl:iota 5) '--){\n}
            {sp}(0 -- 1 -- 2 -- 3 -- 4){\n}
            {sp}> (qtl:split-sequence-if-not #'alphanumericp "3/14/1592"){\n}
            {sp}("3" "14" "1592")
